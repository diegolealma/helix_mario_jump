# YouFace — conceito do personagem

Folha conceitual criada para a branch `youface`.

Ordem dos oito quadros:

1. parado;
2. início do salto;
3. subindo;
4. ápice;
5. caindo;
6. mergulho giratório;
7. aterrissagem;
8. rosto ausente / TV sem sinal.

A área verde oval é uma máscara reservada para a textura da face capturada em
tempo real. O vídeo completo da câmera não deve ser desenhado no jogo.

Arquivos:

- `youface-sprites-chroma.png`: saída original sobre fundo magenta;
- `youface-sprites-concept.png`: versão com fundo transparente.

Este sprite é uma folha de conceito. Antes da integração final, os quadros
devem ser normalizados para células do mesmo tamanho e exportados em resolução
pixel-art apropriada ao renderizador do jogo.
