# Referências dos sprites

Os sprites binários desta pasta foram extraídos individualmente da página
de sprites de *Super Mario World* para SNES mantida pelo SNESMaps:

- https://www.snesmaps.com/maps/SuperMarioWorld/sprites/SuperMarioWorldSprites.html
- `fire_flower.gif`: Fire Flower
- `koopa_green.gif`: Koopa verde
- `koopa_red.gif`: Koopa vermelha
- `shell_green.png`: casco verde
- `shell_red.png`: casco vermelho

Super Mario e seus personagens pertencem à Nintendo. Estes arquivos são
utilizados como referência visual em um projeto de fã.
